== Description ==

This plugin gives support for the jQuery plugin, jScrollPane.

To use, navigate to Dashboard->Settings->WP jScrollPane.
Most questions can be answered by going to http://jscrollpane.kelvinluck.com/

== Installation ==

1. Upload this plugin to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.

== Frequently Asked Questions ==

= Why am I seeing the normal scrollbars and the custom scrollbars? =

Because of the Wordpress Admin Bar. Either log out, or view it in another browser.
